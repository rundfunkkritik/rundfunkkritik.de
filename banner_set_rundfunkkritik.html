Banner-Set – Rundfunkkritik.de

6 fertige Banner in verschiedenen Formaten. So machen Sie Screenshots:

Windows:

  1. Diese HTML-Datei im Browser öffnen (Doppelklick genügt)
  2. Rechtsklick auf den gewünschten Banner → Untersuchen (F12)
  3. Im Inspektor das Banner-Element anklicken → Rechtsklick → Screenshot aufnehmen
  4. Alternativ: Snipping Tool (Win + Shift + S) → Banner-Bereich markieren

Mac:

  1. HTML-Datei im Browser öffnen
  2. Cmd + Shift + 4 → Bereich um den Banner markieren
  3. Screenshot landet automatisch auf dem Desktop
Tipp für pixel-perfekte Screenshots: Browser-Zoom auf 100% stellen (Strg/Cmd + 0), dann Screenshot machen. So stimmen die Abmessungen exakt.
